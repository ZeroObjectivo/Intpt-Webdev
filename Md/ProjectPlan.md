Project Description:

Reference parent: https://umak.edu.ph

The goal of this project is to create a dynamic web application that serves as a platform for the UMak Heron’s Community to post about and interact with various topics as UMak students. This application has these features:

Main Dashboard -> where all users(in public), without logging in yet, can see the collection of contents from different tags, like a social media platform, where there are upvotes, dislikes, and comments.

Login/Signup -> only users from @umak.edu.ph are the only one can create an account

Account Verifier (can use google console api for this the add validations like consent screen)
In this application, there are sections
User Dashboard

The application will have different content based on tags

Creation

Students can create content and they will be asked which tag their post belongs to.
Supports Import/Upload Images, Video, Links, and embed

They will have an option if they want to allow commenting or not on their post
Machine learning can also identify the tag based on the user's content input, but it will create 
complexity.

Tag = Lost and Found
Tag = Heron Business (Buy/Sell)
Tag = General
Tag = Question
Tag = Events

Students can use hashtags for topics or concerns
Viewing

Students can filter out with tag selection. They can select to view only 1 tag, add more tags to view, or select all

Students can search for content from a keyword, and a list of items matching the keyword will appear.

Selecting only 1 tag has a feature to have a dynamic viewing in which the view of contents is changed depending on the tag

Lost and Found

By block viewing of the lost items, and can comment to notify the content creator that someone has messaged them through the app
Heron Business

The content card is different and is more business/selling focused, in which there is a buy option on the contents, and price labels (add)

General tag has the normal content, this is the default tag set for all

Question tag, can see different question list card from the other users

Lastly is the events, in which filtered out all the events happening in the UMak from the content created by other users

Interact
Students either upvote and downvote or uplike and dislike content, like how Reddit works
Students can also comment with filtered-out words to keep the internet etiquette.

COOP Store
Which the students will be able to see the available stocks or items that the UMak Cooperatives has

Books
Uniforms
Merch
Miscellenuous

Scholarship Available/offered, in which is just a panel card per scholarship available can be seen, and if selected, it will open up as a floating card with the description, eligibility, requirements, and when the application is usually due.
